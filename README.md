# Практическая работа №5 — Отладка приложения «Галактики»

### Обнаруженные ошибки
1. **Неправильный тип свойства**: `GalaxyType` объявлен как `object`, из-за чего невозможно получить доступ к `MyGType`.
2. **Опечатка в операторе `switch`**: использовался `case 'l'` (строчная L) вместо `case 'I'` (заглавная I) для типа "Irregular".

### Использованные средства отладки Microsoft Visual Studio
- **Точки останова (Breakpoints)** — для остановки выполнения в цикле `foreach`.
- **Подсказки по данным (Data Tips)** — наведение курсора на переменную `theGalaxy.GalaxyType` для просмотра её содержимого.
- **Пошаговое выполнение (F11 — Step Into)** — для входа в конструктор `GType` и анализа логики `switch`.
- **Окно «Локальные» и «Стек вызовов»** — для отслеживания значений переменных и последовательности вызовов.
- **Анализ ошибок компиляции** — при попытке доступа к `.MyGType` через `object`.

### Результат
После исправления:
- Свойство `GalaxyType` объявлено как `GType`.
- Исправлена опечатка: `case 'I'`.
- Приложение корректно выводит типы всех галактик.
