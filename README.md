# Практическая работа №5 — Отладка приложения «Фибоначчи»

## Цель
Исправить логическую ошибку в функции вычисления 5‑го числа последовательности Фибоначчи с использованием встроенных средств отладки Microsoft Visual Studio.

## Использованные способы отладки Microsoft Visual Studio

- **Точки останова (Breakpoints)** — для приостановки выполнения программы на ключевых строках (`int result = Fibonacci(5);` и внутри цикла).
- **Пошаговое выполнение**:
  - **Step Into (F11)** — для входа в функцию `Fibonacci`.
  - **Step Over (F10)** — для пошагового выполнения кода без входа в вложенные вызовы.
- **Окно «Локальные» (Locals Window)** — для наблюдения за значениями переменных (`n`, `n1`, `n2`, `sum`, `i`) во время выполнения.
- **Команда «Продолжить» (Continue, F5)** — для быстрого перехода от одной точки останова к следующей при многократном проходе цикла.
- **Анализ состояния перед возвратом из функции** — с помощью точки останова на строке `return`.
