## Индивидуальное задание — Вариант №4: Расчёт дохода по вкладу

### Цель
Разработать WPF-приложение для расчёта дохода по вкладу с выбором схемы начисления процентов: простые или сложные.

### Использованные средства отладки Microsoft Visual Studio

При разработке и тестировании приложения были использованы следующие инструменты отладки:

- **Точки останова (Breakpoints)** — для остановки выполнения в обработчике кнопки `btnCalculate_Click`.
- **Пошаговое выполнение (F10, F11)** — для анализа логики расчёта.
- **Окно «Локальные» и «Авто»** — через меню **Отладка → Окна → Локальные / Авто** для просмотра значений переменных.
- **Обработка исключений (`try-catch`)** — для корректной работы при некорректном вводе данных.
- **Инспекция переменных через подсказки (Data Tips)** — наведение курсора на переменные во время отладки.
