# Практическая работа №5 — Отладка приложения «Буквы»

### Цель
Практика использования средств отладки Visual Studio: точек останова, пошагового выполнения, окон «Локальные», «Стек вызовов», подсказок по данным и «Выполнить до щелчка».

### Использованные средства отладки Microsoft Visual Studio
- **Точки останова (Breakpoints)** — на строке `name += letters[i]`.
- **Пошаговое выполнение**:
  - **F10 (Step Over)** — для выполнения без входа в методы.
  - **F11 (Step Into)** — для входа в метод `SendMessage`.
- **Подсказки по данным (Data Tips)** — наведение на переменные `letters`, `name`, `a`.
- **Окно «Локальные» (Locals)** — для просмотра значений переменных в текущем контексте.
- **Стек вызовов (Call Stack)** — для отслеживания последовательности вызовов (`Main` → `SendMessage`).
- **«Выполнить до щелчка» (Run to Click)** — для быстрого перехода к `Console.WriteLine`.

### Результат
Программа корректно выводит имена и счётчики. Ошибок в коде нет.
